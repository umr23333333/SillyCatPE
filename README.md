### 介绍

内置各种常用工具

内置Office2007

内置上网功能

内置声卡驱动

内置影音播放及迅雷下载功能

内置向日葵远程控制

### 食用方法

自行[下载](https://d.umr2333.com/SillyCatPE.iso)刻录即可
